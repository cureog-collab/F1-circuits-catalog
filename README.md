# F1 Circuits Catalog
I built this project to learn web development (HTML, CSS, and JavaScript) by combining it with my personal love for Formula 1.

## Project Overview
A lean, intuitive web app designed to explore maps and details of Formula 1 circuits worldwide. The project focuses on smart data flow handling: seamlessly combining a Live API with internal custom data to accurately recreate the roster of historical, current, and future circuits.

## Dependencies
This project prioritizes minimalism and a clean, modern framework without relying on heavy external libraries.
* **Data Provider:** [Jolpica F1 API](https://api.jolpi.ca/) (Ergast API fork).
* **Typography:** Titillium Web & Work Sans (Google Fonts).
